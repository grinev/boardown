## Release v0.5.1

### Board

- **Create a task from an epic** — The epic dialog's Tasks section gained a plus button that creates a task in that epic without leaving the dialog.
- **Finished releases are read-only** — A task in a finished release opens as plain text with no editable controls, instead of accepting clicks that silently fail, and a finished release is no longer offered as a destination when moving a task.
- **Full release name on the board** — A long release name is shown in full in the board header; the description gives up the space instead.

Full changelog: https://github.com/grinev/boardown/compare/v0.5.0...v0.5.1
